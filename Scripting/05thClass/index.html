Sistemas Operativos — Aula: case & globs (Bash)

Decisões múltiplas elegantes com case + seleção de ficheiros com padrões.

Objetivos Setup Resumo Exemplos Exercícios Recursos Avaliação

Objetivos de Aprendizagem

  • Dominar a sintaxe de case no Bash.
  • Aplicar globs para casar padrões e filtrar ficheiros.
  • Distinguir globbing, brace expansion e extended globbing.

Setup rápido

git clone https://github.com/<org>/<repo>.git
cd <repo>
# (Opcional) instalar shellcheck
sudo apt-get install -y shellcheck

Resumo: `case` e *globs*

Sintaxe `case`

case "$var" in
  padrao1) comando1 ;;
  padrao2|padrao3) comando2 ;;
  *.txt) comando_para_txt ;;
  *) comando_default ;;
esac

*Globs* essenciais

PadrãoSignificadoExemplo
*0+ caracteres*.txt
?1 caractere?.c
[abc]um de a/b/cfile[12].log
[a-z]intervalodata_[a-z].csv
[!0-9]nega dígitosfile[!0-9].txt
[[:digit:]]classe POSIX*[[:digit:]].md

Nota: {a,b} é brace expansion, não globbing. Para extended globs use shopt -s extglob.

Exemplos

Menu com `case`

#!/usr/bin/env bash
set -euo pipefail
PS3="Escolha: "
select opt in "Listar .txt" "Contar .sh" "Sair"; do
  case "$REPLY" in
    1) ls -1 -- *.txt 2>/dev/null || echo "Sem .txt";;
    2) wc -l -- *.sh  2>/dev/null || echo "Sem .sh";;
    3) echo "Até já!"; break;;
    *) echo "Inválido";;
  esac
done

Validação por padrão

read -rp "Ficheiro: " f
case "$f" in
  *.csv) echo "CSV";;
  *.tsv|*.tab) echo "TSV";;
  *[[:space:]]*) echo "Aviso: espaços no nome";;
  *) echo "Desconhecido";;
esac

Extended globs

shopt -s extglob
case "${1,,}" in
  @(start|run|go))   echo "Iniciar";;
  @(stop|halt|end))  echo "Parar";;
  *)                 echo "Ajuda: start|stop";;
esac

Exercícios

  1. Filtro por extensão: listar *.{txt,md} e contar por tipo.
  2. Normalizador: detetar nomes com espaços/maiúsculas e propor renomeação.
  3. CLI com case: aceitar --full, --inc, --help.
  4. Validador: padrões com classes POSIX e case.
  5. Relatório: contagens para *.sh, *test*, *.{png,jpg}.
